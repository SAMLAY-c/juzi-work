- 卖的不是“功能”，而是**单位线索处理效率**。
    
- 价值并不依赖“转化率变好”这种黑箱指标，而是**白纸黑字的成本结构优化**： **同样 20,000 leads，少用 42 个销售，仍能跑完链路**。
    

  

基于上面表格的整理：

|   |   |   |   |
|---|---|---|---|

|转化率|5%|5%|保持不变（关键假设）|

**核心结论**：在人效 3×、leads 与转化率不变的前提下， 你用 **4 万/月** 的产品成本，换来客户 **≈25 万/月** 的净节省， **ROI ≈ 1:6.25**（站得住）。

  

### 在谈判时的建议性策略

1. **不要**让客户把焦点放在“Agent 的功能”上，而是“节省多少销售人力”。
    
2. **不要**被带去“转化率提升”的对赌，先卖**确定性 ROI**。
    
    1. **不承诺成交率，只承诺处理效率** → 这是所有效果分成里最容易被客户搞事的一点，要坚决钉死。
        
3. **明确划线**：“Agent 处理数”就是你收费的基石，转化率提升是 bonus。
    
4. **把客户的成本结构可视化**，让对方自己说出“这单划算”。
