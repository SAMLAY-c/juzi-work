# day1-1-句子互动是一家什么样的公司
# 基本信息
有着非常深厚的技术背景
技术出身、拥有顶级开源项目背书、早在 AI 爆发前就开始研究“人机对话”**的极客团队。
#  基因和感悟
- - **感悟：** 创始人意识到“高光和至暗永远相伴”，必须时刻警惕黑天鹅。
- 最先入局的自动化+时代的浪潮；促进了
- ai点燃了这把rpa 的火

七年前他们就在做对话 AI，那时候太早；现在 OpenAI 把路铺平了，加上他们 7 年的技术积累，终于等来了风口。

- **真正的驱动力不是产品，而是成长。**
    
- **成长 = 学习 + 训练（实战）。** 光看书不行，光干活不思考也不行，必须在炮火中复盘。

1. **底色极客：** 它是写代码出身的，有全球顶级的开源技术基因（Wechaty）。
    
2. **极度务实：** 即使有高大上的 AI 技术，为了生存也愿意弯腰捡硬币，做枯燥的自动化（RPA）。
    
3. **长期主义：** 7 年前就在赌 AI 对话，中间哪怕没风口也坚持熬着，终于熬到了 ChatGPT 时代，实现了“RPA（手）+ AI（脑）”的完美闭环。
    
4. **组织强悍：** 这是一支打不死的团队，经历过行业毁灭性打击（双减），却能逆势翻盘，并且愿意培养自己人。

注重实战，注重经验，注重积累
一个极为广阔的平台与发展前景
# day1-2-句子互动公司文化 | 价值观

| 维度      | 关键词            | 对做事的要求                                                                              |
| ------- | -------------- | ----------------------------------------------------------------------------------- |
| **对客户** | **客户第一**       | **不只做传声筒**。不是客户说什么就是什么，而是像行业专家一样，帮客户找到达成业务目标的最佳方案（甚至敢于挑战客户的不合理需求）。                  |
| **对自己** | **成年人 / 主人翁**  | **不等待指令**。情绪独立，对事不对人；主动发现问题并推动解决，而不是等着问题爆发。按时交付                                     |
| **对团队** | **真诚 / 协作 / ** | **不隐藏问题**。If it didn't happen on the document, it didn't happen. 保持透明，快速同步，持续学习新技术。 |
| 对时代     | 好奇/            | 持续学习新技术。持续创新，把复杂变简单。                                                                |
